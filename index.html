Current status

The Stardance AI 3D Print Model Generator is an in-development web project that will let users enter creative text prompts and receive simple, 3D-printable models (STL) ready for download. This site is the public project home and daily devlog — check back to see progress and new features.

How this will work (overview for visitors)

  • Type a creative prompt describing a small emblem, badge, or trinket.
  • The site will convert text into parametric model rules and render a 3D preview.
  • When ready, users will be able to export the design as an STL file for 3D printing.
